# Mission: Learn Category Theory

## Why
- Long term I want to be able to sketch the proof of the Yoneda Lemma. I want to understand and be able to reproduce its machinery.
- I want to internalize Category theory so that I can spot universal properties in the wild.

## Success looks like
- I can speak fluently about common and even some uncommon constructions in Category Theory
- I can write proofs in Category Theory
- I can apply Category Theory to some fields that I already know the basics in: Calculus, Group Theory, Number Theory, Geometry, Statistics, Probability Theory
- I want to be able to generalize quickly from topics I already know so that I can more quickly learn in topics I am not yet familiar

## Constraints
- This is long term and by and large for fun. Any applications to the real world are a bonus so there is no particular time horizon.

## Theorem toolkit
- Build up a running, citable catalogue of useful theorems and lemmas as we go — see [reference/THEOREMS.html](reference/THEOREMS.html). Headline target: a full sketch of the **Yoneda Lemma**.

## Out of scope
- No particular limits in scope as of yet.
